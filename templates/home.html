GitHub Profile Analyser Web-Application


What will our web app do :

  • As the name suggests, our web app analyses the user’s GitHub profile.
  • After analysing the user’s github profile, it gives important details about the user’s repositories, such as how many forks, clones have been made on each repository.
  • Apart from that, it also shows how many people (unique and total) have visited the user’srepositories.

Why do we need GitHub Profile Analyser?

The GitHub Profile Analyser informs the user about the number of people visiting his/herrepositories, which helps him/her in recognising the popularity of a repository.
This helps theuser in deciding or setting up the priorities among these, as repositories that are more popularrequire more attention.
The profile analyser also tells how many forks and clones have been made since the user has made a particular repository.
This helps the user in understanding theneed of the project among the developers.
